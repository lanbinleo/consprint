# Source Files

This folder is the single home for preserved AP Psychology source material.

- `keyterms.md`: canonical unit, topic, and concept skeleton.
- `unit0.md`, `unit1.md`: local notes used for enrichment.
- `AP-Psychology-Notes.opml`: richer note fragments.
- `unit 2.txt` through `unit 5.txt`: backup/cross-check source files.
- `ai-enrichment.compact`: compact AI enrichment output before import.

Runtime databases live directly under `data/`. AI run logs live under `data/logs/`.
